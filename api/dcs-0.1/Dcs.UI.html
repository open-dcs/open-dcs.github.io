Dcs.UI

Description:

Helper utilities for the UI namespace, mainly an extension of others in the core or application areas.

Content:

Interfaces:

  • Drawable - An object that can be drawn in a Cairo context
  • PluginControl
  • Settable - Implementations of this interface will provide a relevant settings menu and connect to it
  • SettingsWidget - A graphical interface e,ement that is used to set the properties of another element.
  • Widget - Interface for all widgets.
